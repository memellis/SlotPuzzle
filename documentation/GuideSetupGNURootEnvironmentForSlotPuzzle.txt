This guide lists the steps for setting up GNURoot Debian
for SlotPuzzle development on a Samsung Galaxy Note 4

1. Install GNURoot

2. Launch GNURoot to get a terminal window prompt
   and type the following commands:

# apt-get update # refresh apt-get

# apt-get install git-core # install git

# apt-get install ssh

# apt-get install nano

# apt-get install wget

# apt-get install unzip

# apt-get install xclip

# cd /usr/local/src

# git clone https://github.com/memellis/SlotPuzzle

# cd SlotPuzzle

# git config --global user.email "you@example.com"

# git config --global user.name "Mark Ellis"

# ssh-keygen -t rsa -b 4096 -C "memellis@gmail.com"

# eval "(ssh-agen -s)"
